# World Cup Historical Analysis (1930–2022)

Exploratory data analysis of FIFA World Cup matches using Python and pandas.

## Questions answered

- Which team won the most matches in history?
- Which World Cup had the most goals?
- What was the biggest victory margin?
- Which team received the most red cards?
- How many matches ended 0–0?
- Which World Cup had the highest total attendance?
- Which team lost the most matches?
- What percentage of knockout matches were decided by penalties?
- Is there a correlation between yellow cards and goals?

## Stack

- Python
- pandas
- Jupyter Notebooks

## Dataset

[FIFA World Cup Matches 1930–2022](https://www.kaggle.com/) — 964 matches, 44 columns.

## Output

- `output/wc_summary.csv` — key metrics summary
- `output/wc_by_decade.csv` — goals and matches by decade
